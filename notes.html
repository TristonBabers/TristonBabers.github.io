Debugging Notepad

This is pretty much my goto for debugging, especially for gdb as that has a lot of useful, but obscure, commands

Click Me! 
<<<<<<<< GCC >>>>>>>>

*The GCC compiler itself will give warnings that can be used to address quick fixes.

gcc -Wall -Wextra (Program.c)

**Also use -g flag for better debugging with gdb!

<<<<<<<< GDB >>>>>>>>

*gdb can be used to debug programs line by line
*KNOWN ISSUE - GDB contantly gets stuck when stepping, but if u use next, lose your functions too.

gdb [FILE EXECUTABLE] - launches GDB.
b main - breakpoint at main
run - starts program
s - steps
n - next (BAD)
c - continues
q - quits

GDB [EXTRA] -=-=-=-=-=-=-=-=-
gdb ./poll_lookup
lay src
lay regs
b node_lookup.s:27
b print_info.s:42

### TO PRINT REGISTERS:
p $r4
info reg
bt	// SEg FaulTs //Backtrace

<<<<<<<< DIFFY IN THE MIFFY >>>>>>>>
    ./your-program args > output; our-reference args > ref
    diff -s output ref

<<<<<<<< VALGRIND >>>>>>>>

*Finds memory issues

valgrind --leak-check=yes (./Program) arg1 arg2

My custom .vimrc

I mostly use VS Code now, but this comes in handy sometimes.

" *** HIGHLY RECOMMEND :syntax on ***

" Every File Command
:set backspace=indent,eol,start
:set tabstop=4
:set shiftwidth=4
:set expandtab
:set cpoptions=aABceFsI
:set mouse=a
vnoremap  "hy:%s/h//c
"Safer replace for pu$$ies: vnoremap  "hy:%s/h//gc
:vnoremap <  >gv
:vnoremap  >gv
:inoremap { }$i{

"Remap NumPad to actually work
imap Oq 1
imap Or 2
imap Os 3
imap Ot 4
imap Ou 5
imap Ov 6
imap Ow 7
imap Ox 8
imap Oy 9
imap Op 0
imap On .
imap OR *
imap OQ /
imap Ol +
imap OS -

" Set the filetype based on extension, so you don't spend
" 3 hours trying to fix .vimrc
:filetype on
:au BufNewFile,BufRead *.c set ft=c
:au BufNewFile,BufRead *.cpp set ft=cpp
:au BufNewFile,BufRead *.java set ft=java
:au BufNewFile,BufRead *.sh set ft=sh
:filetype detect
:syntax on

" C or C++ Commands
if (&ft == 'c' || &ft == 'cpp')
    :set smartindent
    :colo evening
    :set number
    :set colorcolumn=80
endif

" Arm Commands
if (&ft == 'asm')
    :set syntax=asm
    :highlight Normal ctermfg=white ctermbg=Black
    :highlight LineNr ctermfg=blue
    :highlight asmIdentifier ctermfg=white
    :highlight asmComment ctermfg=green
:set smartindent
    :set number
endif

" Java Commands
if (&ft == 'java' || &ft == 'sh')
    :set background=light
    :colo zellner
    highlight Normal ctermfg=Black ctermbg=LightGrey
    :highlight LineNr ctermfg=Black
    :set smartindent
    :set number
    :set colorcolumn=80
endif
